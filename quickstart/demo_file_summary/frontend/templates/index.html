Read JSON input and produce status count summary

How to Use

  1. Place your JSON input file in the object store at the configured path
  2. Trigger the workflow via the API or Temporal UI
  3. Check the output file for the status count summary

Example Input:

[
  {"id": 1, "status": "completed"},
  {"id": 2, "status": "pending"},
  {"id": 3, "status": "completed"}
]

Example Output:

{
  "total_records": 3,
  "status_counts": {
    "completed": 2,
    "pending": 1
  }
}
